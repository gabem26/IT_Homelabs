# IT_Homelabs
Hands on practice with Active directory/Azure, DHCP, PowerShell, Troubleshooting, and Linux 

## Lab Environment

- **Domain Controller:** Windows Server 2025 — LAB.local
- **Client Machine:** Windows 11
- **Domain:** LAB.local
